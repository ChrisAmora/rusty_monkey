# Rusty Monkey


## About
Rust implementation of Monkey programming language from the book Writing an Interpreter in Go

## Start the repl
```bash
$ cargo run
Welcome to the Monkey programming language!
>> let f = fn(x,y) {x * y + x}
>> let result = f(5 * 3, 10)
>> result
# 165
```


## Testing
```bash
$ cargo test
```
